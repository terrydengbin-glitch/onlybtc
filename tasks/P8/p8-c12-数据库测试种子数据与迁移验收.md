# P8-C12 数据库测试、种子数据与迁移验收

## 状态

DONE

## 所属 Phase

P8 SQLite 数据库与持久化层

## 任务目标

为数据库层建立测试、种子数据、迁移验收和页面模拟数据，保证 P5 UI 和后续 Pipeline 可以基于稳定数据开发。

## 背景依据

- [开发文档.md](../../开发文档.md)
- [task index.md](../../task%20index.md)

## 实施范围

- 单元测试：Repository、migration、constraints、indexes。
- 集成测试：Run Once 空流程写入、snapshot 生成、Evidence Pack 查询。
- seed demo data：Dashboard、Overview、Evidence、LLM Debate、Alerts、Invalidation、Data Quality、Run Logs、Source Detail、Radar Detail、History Replay。
- migration smoke test。
- 数据脱敏测试。

## 输入

- P8-C01 到 P8-C11。
- P5 UI 页面数据需求。

## 输出

- 测试集。
- seed 数据命令。
- 示例数据库。
- 验收报告。

## 验收标准

- 新环境一条命令生成可演示数据库。
- P5 所有页面均可用 seed 数据渲染。
- migration 从空库到最新版本通过。
- raw response 脱敏测试通过。

## 依赖任务

P8-C01、P8-C02、P8-C10、P8-C11

## 备注

该任务完成后，前后端可以并行用 seed data 开发，不必等真实数据源全部接入。
