# P8-C03 数据源、原始观测与标准化指标表

## 状态

DONE

## 所属 Phase

P8 SQLite 数据库与持久化层

## 任务目标

设计并实现 sources、source_runs、raw_observations、normalized_metrics，用于承接 P1 的全部数据源接入。

## 背景依据

- [开发文档.md](../../开发文档.md)
- [task index.md](../../task%20index.md)

## 实施范围

- sources：source_id、source_group、method、preferred_source、fallback_source、update_frequency。
- source_runs：每次抓取任务、状态、耗时、错误、run_id。
- raw_observations：原始响应、hash、observed_at、ingested_at、redaction_status。
- normalized_metrics：标准化字段、单位、时区、parser_version、validation_status。
- raw response 必须支持脱敏和大小限制。

## 输入

- P1 数据源配置 Schema。
- P8-C02 migration 体系。

## 输出

- 数据源注册相关表。
- 原始观测表。
- 标准化指标定义表。
- Repository 基础接口。

## 验收标准

- 任意数据源可以注册。
- 一次抓取能同时记录 raw 与 normalized 结果。
- raw hash 可用于去重。
- Source Detail 页面需要的字段可查询。

## 依赖任务

P8-C02、P1-C01
