## 判断逻辑

### 风险资产

```text
Nasdaq / S&P 500 / Russell 2000 / Dow Jones 上行
=> risk-on
=> 对 BTC 偏多
```

其中：

- Nasdaq：科技与高 beta 风险偏好。
- S&P 500：整体风险资产风向。
- Russell 2000：流动性与小盘风险偏好，更敏感。
- Dow：传统蓝筹，权重较低。

### Gold

Gold 不能简单按“涨=BTC 多/空”处理，必须结合上下文：

```text
Gold 上涨 + DXY 下跌 + real yield 下行
=> 可能是实际利率下行/美元走弱
=> BTC 偏多

Gold 上涨 + VIX 上升 + OFR FSI 上升
=> 可能是避险冲击
=> BTC 不一定受益，偏 mixed/risk-off
```

因此 `gold` 初始设为 `mixed`，进入 Evidence 解释，不作为单边强信号。

### Oil / WTI / Brent

油价也不能简单单边处理：

```text
Oil 上涨 + 风险资产上涨
=> 需求/增长预期改善，BTC 中性偏多

Oil 快速上涨 + 通胀 surprise / 美债收益率上行
=> 通胀压力上升，降息预期受压
=> BTC 偏空

Oil 快速上涨 + VIX/OFR FSI 上升
=> 地缘风险或供给冲击
=> BTC 偏空或 mixed
```

因此 WTI 初始设为 `mixed`，并输出 `energy_pressure` 风险标记。

## Evidence 要求

宏观雷达必须解释：

- 当前偏多/偏空主要来自哪个子因子。
- Gold / Oil 是否只是辅助 evidence。
- 是否存在“风险资产上涨但油价也大涨”的冲突。
- TradingView 与 FRED fallback 是否存在多源冲突。
