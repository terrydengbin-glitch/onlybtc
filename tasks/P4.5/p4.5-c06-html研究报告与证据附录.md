## 所属 Phase

P4.5 Radar Scored Analyst Writer

## 任务目标

生成 P4.5 独立 HTML 报告，以最终文章和 4 个分析员文章为主，scored evidence 和 JSON 放在附录。

## 实施范围

- 顶部展示 run lineage。
- 主体展示最终综合文章。
- 分区展示 4 个分析员文章。
- 附录展示 scored metric evidence 表。
- 支持按 positive/zero/negative/unavailable 过滤。
- 保留原始 JSON 折叠区。

## 验收标准

- HTML 对人类可读。
- 正文不显示大段机器 JSON。
- evidence_id 能在正文和附录之间对应。
