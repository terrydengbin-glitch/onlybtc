# P4.5-C02 Scored Evidence Pack Builder

## 状态

DONE

## 所属 Phase

P4.5 Radar Scored Analyst Writer

## 任务目标

从 P3-C16 输出的 scored metric evidence 构造 P4.5 Analyst Evidence Pack，按 4 个分析员拆分 Radar 模块。

## 实施范围

- 读取 P3 scored evidence。
- 按 Radar module 聚合模块趋势、模块分数、模块质量。
- 按 4 个分析员角色切分 evidence。
- 保留 evidence_id、metric_score、metric_explanation、score_reason。
- 输出 deterministic JSON，供 Analyst Writer 使用。

## 验收标准

- 4 个分析员均能拿到自己的模块 evidence。
- 每个 evidence 都可追溯到 P1/P2/P3 run lineage。
- 不再依赖旧 P4 debate/judge evidence pack。

## 完成记录

已完成 P4.5 Scored Evidence Pack Builder：

- 新增 `backend/src/onlybtc/p45/evidence_pack.py`。
- 从 P3 `p3_scored_metric_evidence` 与 `p3_scored_radar_module` 读取数据。

  - `module_id=p45_analyst_evidence_pack`
