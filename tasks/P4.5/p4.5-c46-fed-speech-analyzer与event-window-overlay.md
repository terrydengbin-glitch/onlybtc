# P4.5-C46 / Fed Speech Analyzer 与 Event Window Overlay 解释层

## 状态
DONE

## Phase
P4.5 报告解释层 / Event Window 语义分析层

## 背景

Event Window 是独立于 14 个 radar 的事实事件与政策冲击层。Fed 官员发言、FOMC 声明、纪要、SEP、新闻稿等文本会影响市场预期，但它们不能直接改写 BTC 趋势分数。

本任务使用 DeepSeek API 对官方文本和高可信 shock headline 做语义分类，只输出政策语气、相关性、置信度和解释文案。所有交易权限修改仍由 Event Window overlay 状态机裁决，所有 BTC 趋势判断仍由 BTC response / radar / cockpit 链条裁决。

## 目标

新增 Fed Speech / Official Text Analyzer：

```text
speaker_weight
policy_relevance
tone
tone_confidence
tone_shift_vs_baseline
requires_human_review
policy_topics

```

## DeepSeek 使用边界

使用现有 LLM provider / DeepSeek 配置：

```text
DEEPSEEK_API_KEY
deepseek model setting
timeout / retry / cache / hash dedupe
```

DeepSeek 可以：

```text
1. Fed / official text tone classification
2. policy topic extraction
3. shock/news category classification
4. human-readable warning summary
5. audit explanation for why overlay was or was not escalated
```

DeepSeek 不可以：

```text
1. 直接输出 BTC bullish / bearish score
2. 覆盖 actual / consensus / nowcast 的数值判断
3. 在 ambiguous / data_dependent 文本中强行判定 hawkish 或 dovish
4. 直接触发 trade_permission_modifier
5. 直接改 radar score、btc cockpit score 或 timescale score
```

## Daemon / LLM Trigger

LLM 不常驻逐秒调用，只由 daemon 在 material change 时入队：

```text
new Fed speech / testimony / press release
FOMC statement / minutes / SEP text hash changed
trusted shock headline with sufficient source quality
official text conflicts with prior baseline
human-readable alert summary required
```

同一 source text hash 不重复调用；低置信输出必须标记：

```text
requires_human_review = true
```

## 审计规则

```text
tone_confidence < 0.70:
  cannot trigger fed_tone_shift

policy_relevance == low:
  cannot escalate emergency_level

tone in [ambiguous, data_dependent]:
  reason can mention uncertainty
  cannot force hawkish/dovish

DeepSeek unavailable:
  analyzer_status = degraded
  deterministic event overlay still runs
```

## DoD

- [x] 支持 `hawkish|dovish|balanced|ambiguous|data_dependent|not_policy_relevant`。
- [x] speaker_weight 按 Chair、Vice Chair、NY Fed、Governor、voter、non-voter 分层。
- [x] `tone_confidence < 0.70` 不得触发 `fed_tone_shift`。
- [x] DeepSeek 输出必须带 `source_url`、`evidence_excerpt` 或 `llm_summary`。
- [x] DeepSeek 输出不得包含 BTC 多空分数或交易方向。
- [x] 同一文本 hash 去重，避免 daemon 重复调用。
- [x] DeepSeek 不可用时，Event Window 状态机仍可运行，并显式记录 degraded。
- [x] 审计 HTML 能展示 Fed speech analyzer 是否遵守边界。
