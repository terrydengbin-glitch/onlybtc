# P4.5-C03 指标一句话解释词典与动态说明生成

## 状态

DONE

## 所属 Phase

P4.5 Radar Scored Analyst Writer

## 任务目标

为每个指标生成适合 LLM 读取的一句话中文解释，帮助 P4.5 对齐指标含义、方向规则和评分原因。

## 实施范围

- 建立 `metric_explanation` 词典。
- 覆盖所有 P2 Radar 指标。
- 解释内容包含：所属主题、指标含义、为何影响 BTC、当前值如何解读。
- 对 zero/unavailable/fallback 给出不同说明。
- 支持动态拼接当前值、方向、分数和质量。

## 验收标准

- P4.5 evidence 中每个指标都有中文解释。
- 解释不包含交易指令。
- LLM prompt 不需要猜指标含义。

## 完成记录

已完成 P4.5 指标解释词典与动态 brief：
