# P4.5-C05 Final Writer 综合研究文章生成

## 状态

DONE

## 所属 Phase

P4.5 Radar Scored Analyst Writer

## 任务目标

基于 4 个 Analyst Writer 输出生成最终中文研究文章，不再执行 LLM 辩论、裁判或交叉质询。

## 实施范围

- 读取 4 篇分析员评论。
- 读取模块分数、正/负/零分分布和关键 evidence。
- 生成最终文章：
  - 核心结论
  - 四大板块共振与分歧
  - 正分指标
  - 负分指标
  - 零分/中性观察项
  - 24h / 3d / 7d 观察路径
  - 数据质量与边界
- 输出结构化 JSON 与 Markdown/HTML 友好正文。

## 验收标准

- 最终文章可独立阅读，像专业研究评论而非审计日志。
- 最终文章引用 4 个分析员结论和关键 evidence。
- 数据质量说明不淹没趋势分析正文。
