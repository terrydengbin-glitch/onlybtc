## 背景

报告层不能继续输出单因子因果：

```text
SOFR 高，所以 BTC 看空
Fed balance sheet 高，所以 BTC 看多
ON RRP 下降，所以 BTC 看多
```

v2.1 报告层需要解释净流动性、repo funding pressure 与 BTC response 的组合关系。

## 目标

让 P4.5 报告正确消费 `dollar_liquidity.v2.1` 结构化 payload，并输出趋势确认/反证解释。

```text
美元净流动性边际改善，但 BTC 没有跟随，说明流动性顺风尚未被价格吸收。
```

```text
TGA 回落和准备金回升缓解了短期流动性压力，但 ON RRP 已接近耗尽，后续边际改善需要看 TGA 与 reserves。
```

```text
SOFR 相对 IORB 没有上冲，repo funding 暂未构成 BTC 的主要压力源。
```

```text
美元流动性逆风下 BTC 仍保持正回报，说明币圈内部买盘强于宏观流动性压力。
```

## 禁止文案

```text
SOFR 高，所以 BTC 看空。
Fed balance sheet 下降，所以 BTC 看空。
ON RRP 下降，所以 BTC 看多。
TGA 高，所以 BTC 必跌。
```
