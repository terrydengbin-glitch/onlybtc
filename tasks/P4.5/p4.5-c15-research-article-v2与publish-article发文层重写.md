# P4.5-C15 Research Article v2 与 Publish Article 发文层重写

## 状态

DONE

## 所属 Phase

P4.5 Radar Scored Analyst Writer

## 任务目标

将 P4.5 v2 的研究正文和发文层从旧模板彻底切到 `final_view` 契约，生成真正面向人类阅读的中文研究结论和平台发文版本，禁止 raw evidence_id、run_id、Python dict、小数评分和英文方向词污染正文。

## 背景

当前 P4.5 HTML 虽然已经在顶部新增 v2 决策区，但“最终综合研究文章”仍然继承旧模板，存在：

- 继续输出 `p3-score-...`
- 输出 Python dict 风格方向分布
- 继续写“核心判断为偏空”
- 出现 deterministic fallback 说明
- 与 `decision_card.direction` 和 `publish_article` 方向冲突

这说明 v2 只是加在旧 HTML 前面，研究正文还没有真正被 v2 契约接管。

## 实施范围

1. Research Article v2
   - 新增或升级：

```json
{
  "research_article": {
    "schema_version": "p45.research_article.v2",
    "title": "...",
    "executive_summary": "...",
    "core_view_narrative": "...",
    "horizon_narrative": "...",
    "support_pressure_narrative": "...",
    "invalidation_narrative": "...",
    "data_boundary": "...",
    "safe_for_human_reading": true
  }
}
```

   - 必须使用 `final_view` 作为主结论。
   - 禁止继续使用旧 bearish fallback 文案。

2. Research Article 内容约束
   - 禁止出现：
     - `p3-score`
     - `run_id`
     - `collect_run_id`
     - `schema_version`
     - Python dict 原样文本
   - Evidence ID 只允许出现在 Evidence 附录和 Final JSON。

3. Publish Article 风格升级
   - `style` 改为：
     - `laoma_market_view` 或 `trader_market_view`
   - 禁止出现：
     - 综合分数小数
     - 置信度小数
     - `bullish / bearish / neutral` 英文方向词
     - 内部字段名
   - 必须包含：
     - 清晰观点
     - 观察点
     - 反转/确认条件
     - `$BTC`

4. Publish Article 校验字段

```json
{
  "publish_article": {
    "style": "laoma_market_view",
    "contains_score_number": false,
    "contains_english_direction": false,
    "has_clear_view": true,
    "has_watch_points": true,
    "has_cashtag": true,
    "no_internal_terms": true
  }
}
```

5. LLM 文章与 final_view 的关系
   - LLM 文章作为解释参考。
   - 如果 LLM 文章方向与 `final_view` 不一致，由 P4.5-C13 标记冲突。
   - 本任务只负责在 HTML 中分层展示：
     - canonical research article
     - publish article
     - LLM reference article
     - evidence appendix

## 推荐发文风格示例

```text
$BTC 这里我不看单边强空，也不急着追多。短线有一点承接，1h 成交量和交易所余额变化还算给支撑，但 3 天到 7 天维度压力没消失，ETF 资金还没修复，SOFR 和真实利率仍压着风险资产。

所以老马这边先按震荡偏谨慎看。接下来就盯三件事：ETF 流出能不能收窄，1h 成交量能不能继续放大，主动买盘能不能重新转强。没确认前，不适合重仓追方向，反弹先看压力，回踩看承接。
```

## 验收标准

- `research_article` 使用 v2 契约。
- `research_article` 主结论等于 `final_view`。
- `research_article` 不出现 raw evidence_id / run_id / Python dict。
- `publish_article.body` 不出现 raw evidence_id / run_id / schema_version。
- `publish_article.body` 不出现小数评分和英文方向词。
- `publish_article` 包含 `$BTC`。
- `publish_article` 有明确观点和观察点。
- HTML 中 Evidence 附录仍保留在尾部，不污染正文。
- Contract validation 能检查研究正文和发文层 forbidden content。
