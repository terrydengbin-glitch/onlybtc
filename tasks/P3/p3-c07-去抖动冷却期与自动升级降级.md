# P3-C07 去抖动、冷却期与自动升级降级

## 状态

DONE

## 实现结果

已在 `generate_algorithm_alerts()` 内实现 alert lifecycle。

## 机制

- 使用稳定 `alert_id = alert-{category}`，同类预警复用同一条 `algorithm_alerts`。
- 重复触发不会刷屏，而是更新既有 alert，并追加 `alert_events`。
- 根据等级变化写入状态：
  - `active`
  - `cooling`
  - `escalated`
  - `downgraded`

## 冷却期

```yaml
info: 30m
watch: 60m
warning: 120m
critical: 240m
```

## 验收

- 同类信号连续两次触发时，不新增重复 alert。
- `alert_events` 记录 cooling/escalated/downgraded。
- 冷却时间写入 `cooldown_until`。
